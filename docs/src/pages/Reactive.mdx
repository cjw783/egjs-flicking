# Reactive API

`Reactive API`는 Flicking의 상태를 반응형으로 관리할 수 있는 API입니다. 이벤트 기반이 아닌 상태 기반으로 동작하여 더 직관적이고 효율적인 개발이 가능합니다.

## 주요 특징

- **반응형 상태 기반 API**: 이벤트 대신 상태 변화를 감지하여 UI 업데이트 가능
- **외부 UI 컴포넌트 연계**: mui, antd 등 다양한 UI 컴포넌트와 쉽게 연동
- **프레임워크 지원**: 현재 JavaScript, React, Vue2, Vue3 환경에서 사용 가능하며 추후에 다른 프레임워크도 지원 예정

## 제공되는 상태

| 상태 | 타입 | 설명 |
|------|------|------|
| `isReachStart` | `boolean` | 첫 번째 패널에 도달했는지 여부 |
| `isReachEnd` | `boolean` | 마지막 패널에 도달했는지 여부 |
| `totalPanelCount` | `number` | 전체 패널 개수 |
| `currentPanelIndex` | `number` | 현재 활성화된 패널의 인덱스 |
| `progress` | `number` | 전체 스크롤 진행률 (0-100) |
| `indexProgress` | `number` | 패널 간 진행률 (소수점 포함) |

```js
import Flicking, { connectFlickingReactiveAPI } from "@egjs/flicking";

// Flicking 인스턴스 생성
const flicking = new Flicking("#el");

// Reactive API 연결 - flicking 인스턴스의 반응형 상태들을 가져옵니다.
const reactiveObj = connectFlickingReactiveAPI(flicking);

// 상태들을 직접 꺼내서 사용할 수 있습니다
const { currentPanelIndex, isReachStart, progress } = reactiveObj;

// 현재 상태 값들을 출력
console.log("현재 패널 인덱스:", currentPanelIndex);
console.log("첫 번째 패널 도달:", isReachStart);
console.log("진행률:", progress + "%");

// subscribe 메서드를 통해서 특정 상태 변화를 감지할 수 있습니다.
reactiveObj.subscribe("currentPanelIndex", (nextValue) => {
  console.log("현재 패널 인덱스:", nextValue);
});

reactiveObj.subscribe("isReachStart", (nextValue) => {
  console.log("첫 번째 패널 도달:", nextValue);
});

reactiveObj.subscribe("progress", (nextValue) => {
  console.log("진행률:", nextValue + "%");
});
```

```jsx
import React, { useRef } from "react";
import Flicking from "@egjs/react-flicking";
import { useFlickingReactiveAPI } from "@egjs/react-flicking";

export default function App() {
  const flickingRef = useRef(null);
  
  // Reactive API 훅 사용 - flicking의 상태가 자동으로 reactive하게 관리됩니다
  const {
    currentPanelIndex,
    isReachStart,
    isReachEnd,
    progress,
    moveTo
  } = useFlickingReactiveAPI(flickingRef);

  return (
    <div>
      <Flicking ref={flickingRef}>
        <div className="panel">Panel 1</div>
        <div className="panel">Panel 2</div>
        <div className="panel">Panel 3</div>
      </Flicking>
      
      {/* 상태가 실시간으로 반영되는 UI */}
      <div>현재 패널: {currentPanelIndex + 1}</div>
      <div>진행률: {progress.toFixed(1)}%</div>
      <button disabled={isReachStart}>이전</button>
      <button disabled={isReachEnd}>다음</button>
      
      {/* 특정 패널로 이동 */}
      <button onClick={() => moveTo(1)}>두 번째 패널로 이동</button>
    </div>
  );
}
```

```js
import Flicking, { connectFlickingReactiveAPI } from "@egjs/flicking";

const flicking = new Flicking("#el");
const reactiveObj = connectFlickingReactiveAPI(flicking);

// Prev/Next 버튼과 연동
const prevBtn = document.querySelector(".prev-btn");
const nextBtn = document.querySelector(".next-btn");

prevBtn.addEventListener("click", () => {
  if (!reactiveObj.isReachStart) {
    reactiveObj.moveTo(flicking.index - 1);
  }
});

nextBtn.addEventListener("click", () => {
  if (!reactiveObj.isReachEnd) {
    reactiveObj.moveTo(flicking.index + 1);
  }
});

// 버튼 상태 업데이트
reactiveObj.subscribe("isReachStart", (nextValue) => {
  prevBtn.disabled = nextValue;
});

reactiveObj.subscribe("isReachEnd", (nextValue) => {
  nextBtn.disabled = nextValue;
});
```

```jsx
  const flickingRef = React.useRef(null);
  const {
    isReachStart,
    isReachEnd,
    moveTo
  } = useFlickingReactiveAPI(flickingRef);

  const handlePrev = () => {
    if (!isReachStart) {
      moveTo(flickingRef.current.index - 1);
    }
  };

  const handleNext = () => {
    if (!isReachEnd) {
      moveTo(flickingRef.current.index + 1);

    <div>
      <Flicking ref={flickingRef}>
        {panels.map((panel, index) => (

        ))}
      </Flicking>
      
      <div style={{ display: "flex", gap: "10px", justifyContent: "center" }}>
        <Button
          variant="contained"
          onClick={handlePrev}
          disabled={isReachStart}
        >
          Previous
        </Button>
        <Button
          variant="contained"
          onClick={handleNext}
          disabled={isReachEnd}
        >
          Next
        </Button>
      </div>
    </div>
  );
}
```
